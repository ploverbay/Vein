  #### 图设计的快捷方式
  - 在画布中双击可以在双击位置增加一个节点
  - 选择节点，在键盘上按Delete键（有些键盘是fn+Back）可以快速删除节点
  - 在画布的空白位置按住可拖拽画布
  - 在画布上滚动鼠标中间（或滑动触控板）可以放大缩小画布
